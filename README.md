# Chopp Correto - Privacy Policy

Politica de privacidade do aplicativo Android **Chopp Correto**
(pacote `br.com.prcompany.choppcorretoapp`), publicado por PR Company.

Versao publicada: https://holocaster.github.io/chopp-contador-privacy-policy/

O conteudo da pagina esta em `index.html`, em portugues e ingles.

## Habilitar GitHub Pages

1. No GitHub, abrir Settings -> Pages.
2. Em "Source", escolher branch `main` e pasta `/ (root)`.
3. Salvar. A URL publicada aparece em alguns minutos.
4. Usar essa URL no Google Play Console em
   App content -> Privacy policy.
